# FeederTwin

*Simulation model and control-strategy study for an automatic single-part feeding system*

Имитационная модель и исследование стратегий управления для автоматической системы поштучной подачи мелкоразмерных изделий (вибрационный лоток + машинное зрение YOLOv8 + заслонка с сервоприводом). Проект развивает НИР «Разработка автоматической системы поштучной подачи мелкоразмерных изделий» (НИТУ МИСИС): спроектированная киберфизическая система переводится в вычислительную модель для исследования режимов транспортирования и синтеза стратегий управления.

**Статус: в разработке, этап 1/6.**

## Ключевой результат этапа 1

Рабочая точка спроектированной системы (A = 0,25–0,3 мм, f = 16,7 Гц) лежит в **безотрывном режиме** (коэффициент режима Γ = A·ω²/g ≈ 0,28–0,34). Точечная кулоновская модель показывает, что при вертикальных колебаниях в этом режиме среднее транспортирование отсутствует при tg α ≤ μ; порог режима подбрасывания достигается при f ≈ 30 Гц (A = 0,275 мм) либо A ≈ 0,9 мм (f = 16,7 Гц). Подробности и следствия для проектирования: [docs/stage1_transport_findings.md](docs/stage1_transport_findings.md).

## Quickstart

```bash
python -m venv .venv && .venv\Scripts\activate   # Windows
pip install -e ".[dev]"
pytest                                  # 12 тестов
python scripts/run_transport_study.py   # сводка + 3 графика в reports/figures/
```
