# FeederTwin

*Simulation model and control-strategy study for an automatic single-part feeding system*

Имитационная модель и исследование стратегий управления для автоматической системы поштучной подачи мелкоразмерных изделий (вибрационный лоток + машинное зрение YOLOv8 + заслонка с сервоприводом). Проект развивает НИР «Разработка автоматической системы поштучной подачи мелкоразмерных изделий» (НИТУ МИСИС): спроектированная киберфизическая система переводится в вычислительную модель для исследования режимов транспортирования и синтеза стратегий управления.

**Статус: в разработке, этап 4/6.**

## Ключевые результаты

### Этап 1 · Вибротранспортирование

Рабочая точка спроектированной системы (A = 0,25–0,3 мм, f = 16,7 Гц) лежит в **безотрывном режиме** (коэффициент режима Γ = A·ω²/g ≈ 0,28–0,34). Точечная кулоновская модель показывает, что при вертикальных колебаниях в этом режиме среднее транспортирование отсутствует при tg α ≤ μ; порог режима подбрасывания достигается при f ≈ 30 Гц (A = 0,275 мм) либо A ≈ 0,9 мм (f = 16,7 Гц). Подробности: [docs/stage1_transport_findings.md](docs/stage1_transport_findings.md).

### Этап 2 · Контур контроля единичной подачи

Дискретно-событийная модель контура «камера → детектор → заслонка» на временных параметрах НИР (подтверждение 2 кадра @ 30 fps, привод 40 мс) воспроизводит проектный бюджет коррекции (103 мс до ввода заслонки) и снижает дефекты подачи на ~11% в рабочей точке. Обнаружена **граница устойчивости пороговой стратегии (~5,5 дет/с)**: выше неё заслонка дестабилизирует поток через положительную обратную связь сгущения, а целевые 5 дет/с лежат вблизи границы. Сформировано количественное требование к детектору: средняя уверенность на нарушениях ≥ 0,75. Подробности: [docs/stage2_flow_findings.md](docs/stage2_flow_findings.md).

### Этап 3 · Стратегии управления

Сравнение четырёх законов управления на одинаковых реализациях потока. **Дозирующий выпуск** (заслонка как шлюз с выдержкой технологического зазора) доминирует по Парето: дефекты падают на два порядка (1,4 против 96 на 1000 у базовой) при потере производительности < 4%, граница устойчивости исчезает во всём диапазоне 3–8 дет/с. Адаптивное регулирование вибрации выходит на предельный цикл и втрое снижает дефекты на перегрузке ценой ~30% производительности. Подробности: [docs/stage3_strategies_findings.md](docs/stage3_strategies_findings.md).

### Этап 4 · Многокритериальный выбор

AHP (с контролем согласованности) и TOPSIS реализованы с нуля и верифицированы против эталонов из литературы. Два кейса: выбор стратегии управления на реальных метриках этапа 3 (**дозирующий выпуск, близость 0,852**, лидер при любом весе качества > 0,18) и формализация выбора метода контроля из главы 4 НИР (машинное зрение 0,815; смена лидера лишь при весе стоимости > 0,35). Подробности: [docs/stage4_mcda_findings.md](docs/stage4_mcda_findings.md).

## Quickstart

```bash
python -m venv .venv && .venv\Scripts\activate   # Windows
pip install -e ".[dev]"
pytest                                  # 42 теста
python scripts/run_transport_study.py   # этап 1: модель вибротранспортирования
python scripts/run_flow_study.py        # этап 2: контур контроля, Monte Carlo
python scripts/run_strategies_study.py  # этап 3: сравнение стратегий управления
python scripts/run_mcda_study.py        # этап 4: AHP + TOPSIS, два кейса выбора
```
