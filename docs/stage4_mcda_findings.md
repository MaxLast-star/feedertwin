# Этап 4. Многокритериальный выбор (AHP + TOPSIS): выводы

## Постановка

Реализован модуль многокритериального анализа решений (`mcda.py`):
AHP (веса критериев как главный собственный вектор матрицы парных
сравнений, контроль согласованности по CI/CR с порогом 0,10), TOPSIS
(векторная нормализация, близость к идеальному решению) и анализ
чувствительности (свип веса одного критерия с пропорциональным
масштабированием остальных). Реализация верифицирована тестами против
эталонных примеров из литературы (точное восстановление весов
согласованной матрицы; пример Саати: λ ≈ 3,039, CR ≈ 0,033).

Методы применены к двум реальным задачам проекта.

## Кейс 1. Выбор стратегии управления

Альтернативы и метрики — результаты Monte Carlo этапа 3 (5 дет/с);
сложность реализации — экспертная балльная оценка; износ — доля времени
работы заслонки. Критерии и AHP-веса (CR = 0,043, согласовано):
качество подачи 0,565; производительность 0,262; сложность 0,118;
износ 0,055.

Ранжирование TOPSIS: **дозирующий выпуск 0,852** ≫ адаптивная 0,426 >
без контроля 0,186 > базовая 0,148. Примечательно: базовая пороговая
стратегия заняла последнее место — она сочетает скромный выигрыш
по качеству с полной ценой по сложности и износу.

Анализ чувствительности (mcda_sensitivity.png): дозирующий выпуск
остаётся лидером при любом весе «качества подачи» выше ~0,18 —
вывод робастен к субъективности парных сравнений. Лишь при почти
нулевой значимости качества (< 0,18) лидером становится «без контроля»
(простота побеждает, когда цель системы не важна — вырожденный случай).

## Кейс 2. Выбор метода контроля единичности

Формализация качественного сравнения главы 4 НИР: однолучевой датчик /
световой барьер / машинное зрение. AHP-веса (CR = 0,018): информативность
0,492; гибкость к номенклатуре 0,306; стоимость 0,125; сложность
внедрения 0,078.

Ранжирование TOPSIS: **машинное зрение 0,815** ≫ барьер 0,280 >
однолучевой 0,185 — выбор НИР воспроизводится формальной процедурой.
Чувствительность: машинное зрение остаётся лидером, пока вес стоимости
ниже ~0,35; выше — выгоднее однолучевой датчик. Это превращает
качественный довод НИР в количественную границу: выбор CV оправдан
для всех сценариев, где стоимость не является доминирующим критерием
(вес < 1/3).

## Ограничения

Матрицы парных сравнений отражают суждения одного эксперта (автора);
групповое агрегирование (геометрическое среднее матриц) не применялось.
Балльные оценки кейса 2 — порядковые шкалы, использованные как
количественные; для целей сравнительного ранжирования при анализе
чувствительности это допустимо, но абсолютные значения близостей
интерпретации не подлежат.
