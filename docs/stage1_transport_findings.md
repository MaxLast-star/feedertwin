# Этап 1. Модель вибротранспортирования: выводы

## Постановка

Построена численная модель установившегося движения одиночной детали
(материальная точка, кулоновское трение, неупругое приземление e = 0)
по наклонному лотку, совершающему вертикальные гармонические колебания
s(t) = A·sin(ωt) — по схеме двухвального дебалансного вибратора со встречным
вращением валов из НИР. Полный список допущений — в докстринге
`src/feedertwin/transport.py`.

## Результаты

### 1. Рабочая точка находится в безотрывном режиме

Коэффициент режима Γ = A·ω²/g для проектных параметров НИР:

| A, мм | f, Гц | Γ | Режим |
|---|---|---|---|
| 0,25 | 16,7 | 0,281 | безотрывный |
| 0,275 | 16,7 | 0,309 | безотрывный |
| 0,30 | 16,7 | 0,337 | безотрывный |

Отрыв деталей от лотка (режим подбрасывания) наступает при Γ > 1, то есть
при f ≈ 30 Гц (для A = 0,275 мм) либо при A ≈ 0,9 мм (для f = 16,7 Гц).

### 2. В безотрывном режиме точечная модель не даёт транспортирования

При вертикальных колебаниях и контакте движущая сила вдоль лотка и сила
трения пропорциональны одной и той же величине u(t) = g + s̈(t):

ξ̈ = u(t)·(sin α − μ·cos α·sign ξ̇)

Поэтому при tg α ≤ μ деталь покоится в любой фазе цикла, а при tg α > μ —
неограниченно разгоняется (соскальзывание, нет установившегося режима).
Средняя скорость подачи, управляемая вибрацией, в этой модели возникает
только в режиме подбрасывания: разгон в фазах полёта, торможение трением
в фазах контакта (см. reports/figures/velocity_vs_alpha.png,
velocity_vs_freq.png).

### 3. Следствия для физической системы

Модель уровня материальной точки сознательно не учитывает механизмы,
которые могут обеспечивать подачу в реальной системе при Γ < 1:

- упругие моды лотка и локальные ускорения, превышающие A·ω² жёсткого тела
  (измеренная амплитуда 0,25–0,3 мм — интегральная характеристика);
- микрорельеф резиновой подкладки (выступы создают асимметрию трения,
  аналог анизотропного транспортирования);
- неточечность деталей: кувыркание и микроудары о выступы.

### 4. Рекомендации к проектированию (гипотезы для проверки)

1. Зафиксировать паспортный режим транспортирования: либо подтвердить
   подачу при Γ ≈ 0,3 экспериментально (тогда механизм — п. 3), либо
   перевести рабочую точку в Γ = 1,1–1,5: f ≈ 32–37 Гц при A = 0,275 мм
   или увеличение массы дебалансов.
2. Угол наклона α — слабый рычаг в безотрывном режиме (скорость нулевая
   во всём диапазоне 0–15° при μ = 0,3), но основной рычаг регулирования
   в режиме подбрасывания: на этом будет построена стратегия управления (в)
   этапа 3.
3. Диапазон регулировки угла 0–15° при μ ≥ 0,3 (сталь по резине — реально
   больше) не достигает границы соскальзывания tg α = μ — конструкция
   безопасна от неуправляемого схода деталей. Это положительный вывод
   о запасе устойчивости спроектированного узла регулировки.

## Ограничения

Модель не валидирована экспериментально; коэффициент трения μ —
варьируемый параметр анализа чувствительности (reports/figures/
sensitivity_mu.png). Количественные значения скоростей следует читать
как порядковые оценки; качественные пороги (Γ = 1, tg α = μ) —
строгие следствия принятых допущений.
