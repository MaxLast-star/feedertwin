# Сводный отчёт о результатах

## Что модель сказала о системе, чего не сказал чертёж

Четыре нетривиальных вывода, полученных моделью и отсутствующих
в конструкторской документации НИР.

### 1. Рабочая точка — глубоко безотрывный режим

Коэффициент режима Γ = A·ω²/g ≈ 0,28–0,34 при проектных параметрах
(A = 0,25–0,3 мм, f = 16,7 Гц). Точечная кулоновская модель в этом
режиме не даёт среднего транспортирования при tg α ≤ μ. Следствие:
подача в реальной системе обеспечивается механизмами вне точечной модели
(упругие моды лотка, микрорельеф подкладки), а порог режима подбрасывания
достигается лишь при f ≈ 30 Гц или A ≈ 0,9 мм. Это указывает направление
доработки вибропривода, если потребуется устойчивое управляемое
транспортирование.

### 2. Запас по устойчивости узла регулировки угла

Диапазон наклона 0–15° при μ ≥ 0,3 не достигает границы соскальзывания
tg α = μ — конструкция узла регулировки безопасна от неуправляемого
схода деталей. Положительный конструкторский вывод.

### 3. Граница устойчивости пороговой стратегии ≈ 5,5 дет/с

Реактивная заслонка дестабилизирует поток выше ~5,5 дет/с через обратную
связь скучивания; целевые 5 дет/с лежат у самой границы — запас мал.
Этого ограничения нет в проектной документации.

### 4. Дозирующий выпуск снимает ограничение

Смена режима заслонки с «отсекатель» на «тактируемый шлюз» снижает дефекты
со ~96 до ~1,4 на 1000 при той же производительности и устраняет границу
устойчивости. Главная инженерная рекомендация проекта; не требует новых
механизмов, только посчёт деталей на входе (та же камера).

## Требования к смежным подсистемам (производные)

- **Детектор:** средняя уверенность на классе «нарушение» должна
  превышать порог решения (≥ 0,75); при уверенности ниже ~0,7 контур
  бесполезен.
- **Посчёт деталей (для дозирования):** надёжность ≥ 0,98 даёт остаточные
  дефекты 1–2/1000; метрика для приёмочных испытаний.

## Обоснованность выбора (MCDA)

AHP подтвердил согласованность экспертных приоритетов (CR = 0,043 и 0,018
для двух кейсов). TOPSIS: дозирующий выпуск — лидер (близость 0,852),
устойчивый при любом весе критерия качества выше 0,18; выбор машинного
зрения из главы 4 НИР воспроизведён формально (0,815) с границей
применимости по стоимости (вес < 0,35).

## Ограничения и дальнейшая работа

Модель не валидирована натурно; параметры детектора и потока заданы
экспертно. Развитие: калибровка по экспериментальным данным стенда,
учёт неточечности деталей, оптимизация параметров адаптивной стратегии,
связка «угол наклона → скорость → интенсивность потока» в единый сквозной
расчёт.
