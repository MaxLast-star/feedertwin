# Алгоритмы управления

Контур контроля единичной подачи замыкает цепь «восприятие → решение →
воздействие». Реализованы и сопоставлены три закона управления.

## Контур в целом

```
 поток деталей ─► зона контроля ─► камера (30 к/с) ─► детектор (порог 0,75)
                      ▲                                       │
                      │                                  подтверждение
                 коррекция                                 2 кадров
                      │                                       │
                 заслонка ◄─── привод (40 мс) ◄─── решение стратегии
```

Бюджет времени реакции: 66 мс (2 кадра) + 40 мс (привод) + 50–120 мс
(стабилизация). Модель воспроизводит среднюю латентность ≈ 103 мс
до ввода заслонки.

## Стратегия 1. Базовая пороговая (НИР)

Релейный закон: N = 2 последовательных кадра с уверенностью выше порога
0,75 → команда на ввод заслонки, отсечение лишних деталей.

Свойство: реагирует на уже возникшее нарушение. Обнаруженное ограничение —
**граница устойчивости ≈ 5,5 дет/с**: выше неё блокировка входа порождает
скучивание деталей и положительную обратную связь, и контур начинает
вредить. Целевые 5 дет/с лежат у самой границы.

## Стратегия 2. Адаптивная вибрация

Базовый закон + внешний контур по частоте срабатываний:

```
если (срабатываний ≥ 3 за 2 с):      factor ← max(0,5; factor × 0,8)
если (тишина ≥ 3 с):                 factor ← min(1,0; factor × 1,05)
интенсивность подачи = base × factor
```

Реализует идею НИР (п. 5.4) «при частых ошибках снижать интенсивность
вибрации». Поведение — устойчивый предельный цикл вокруг множителя
0,5–0,65 при перегрузке: дефекты втрое ниже базовых ценой ~30%
производительности. Целесообразна как аварийный режим, не как основной.

## Стратегия 3. Дозирующий выпуск

Смена принципа: заслонка работает как тактируемый шлюз, выдерживающий
минимальный технологический зазор между входами в зону:

```
при входе детали (с надёжностью посчёта p):
    следующий вход разрешён не ранее, чем через headway = 65 мс
```

Поскольку проактивно предотвращает скучивание, а не борется с ним
постфактум, снижает дефекты на два порядка и устраняет границу
устойчивости во всём диапазоне 3–8 дет/с. Детекторная защита сохраняется
как вторая линия.

## Сравнение (5 дет/с, Monte Carlo)

| Стратегия | Дефекты /1000 | Производительность, дет/с |
|---|---|---|
| Без контроля | 100,9 | 5,37 |
| Базовая пороговая | 95,9 | 5,33 |
| Адаптивная вибрация | 57,3 | 3,64 |
| **Дозирующий выпуск** | **1,4** | **5,18** |

## Связь с теорией управления

Базовая стратегия — релейный (двухпозиционный) регулятор; адаптивная —
двухконтурная система с медленным внешним контуром, демонстрирующая
автоколебания при насыщении; дозирующий выпуск — упреждающее (feedforward)
управление темпом. Сопоставление этих типов на единой модели — предмет
направления «Интеллектуальные системы управления».
